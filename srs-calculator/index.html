The Mathematics

This tool computes the Shock Response Spectrum (SRS) by calculating the time-domain response of a Single Degree of Freedom (SDOF) system to the input pulse.

Equation of Motion

For each natural frequency \( f_n \), the system responds according to the SDOF equation of motion:

$$ \ddot{z} + 2\zeta\omega_n \dot{z} + \omega_n^2 z = -\ddot{y}(t) $$

Where:

  • \( z(t) \): Relative displacement
  • \( \omega_n = 2\pi f_n \): Natural angular frequency
  • \( \zeta = \frac{1}{2Q} \): Damping ratio
  • \( \ddot{y}(t) \): Base acceleration (Input Pulse)

Absolute Acceleration

The plot displays the Maximax Absolute Acceleration (Peak Response), calculated as:

$$ \text{SRS}(f_n) = \max | \ddot{x}(t) | = \max | \ddot{z}(t) + \ddot{y}(t) | $$

Integration Method

The system uses a Velocity Verlet numerical integration scheme. This symplectic integrator is chosen for its stability in conserving energy for oscillatory systems. We over-sample the time domain (dt < 1/20th of the highest natural period) to ensure numerical stability and peak-capture accuracy.

Practical Interpretation

Think of SRS as testing your shock against a shelf of tuning forks, each with a different natural frequency. Each point on the SRS curve answers: "If I have a component with this resonance frequency, what G-level will it experience?"

How to Use SRS

  1. Identify your component's resonance ‚Äî e.g., a circuit board at 200 Hz
  2. Read the SRS value at that frequency ‚Äî e.g., 80G at 200 Hz
  3. Qualify your component ‚Äî test it to that level (plus margin, typically 3-6 dB)
  4. Redesign if needed ‚Äî stiffen to shift resonance away from peaks, or add damping

Reading the SRS Curve

  • Peak location: Components with this resonance see the worst response
  • Low-frequency plateau: Slow components move with the input (quasi-static)
  • High-frequency roll-off: Fast components don't "see" the slow shock
  • Higher Q-factor: Less damping ‚Üí sharper, higher peaks

The SRS provides an upper bound estimate, not an exact prediction. Real components have multiple resonances, complex mode shapes, and interactions the idealized SDOF model doesn't capture. Use SRS as a screening tool and specification method ‚Äî it's extremely useful for that purpose, which is why NASA, MIL-STD, and aerospace test specs all use it.
